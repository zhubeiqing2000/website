论文现状
  • MLP/ANN模型做离线批量预测，输入太阳风参数+地磁指数+空间位置，输出电子通量
  • 评估方式：R值、RMSE、真实磁暴事件回放验证
  • 交互方式：无（纯学术研究，无用户界面）
深化方案：多Agent协作的实时预警系统

将离线预测模型升级为由4个独立Agent协作的实时预警产品。

数据采集 Agent
定时拉取NOAA SWPC实时太阳风数据、地磁指数
API轮询 + 数据缓存 + 异常重试
预测推理 Agent
在线模型推理，输出4能量通道×2投掷角通量预测
模型服务化 (Flask/FastAPI) + 批量推理
风险决策 Agent
根据预测通量判定风险等级，触发预警规则
规则引擎 + 阈值配置 + 历史对比
交互推送 Agent
向卫星运营商推送预警信息与操作建议
WebSocket + 消息队列 + 多渠道通知
